# ✈️ Airline Reservation System

A comprehensive **Airline Reservation System** developed as a **Semester Project** using **Python** and **MySQL**. This application streamlines the process of managing flight reservations by providing an intuitive interface for passengers and administrators to handle bookings, cancellations, and flight management efficiently.

---

## 📌 Project Overview

The Airline Reservation System is designed to automate airline booking operations while ensuring accurate data management through a MySQL database. It demonstrates the practical implementation of Object-Oriented Programming (OOP), database connectivity, and CRUD operations in a real-world application.

## 🗄️ Database Setup

This project uses **MySQL** as its database.

### Steps

1. Install MySQL Server.
2. Open MySQL Workbench (or phpMyAdmin).
3. Create a new database.
4. Import the provided SQL file:

```
airline_db.sql
```

5. Update the database credentials in the Python project according to your local MySQL configuration.

---

## ▶️ Installation & Running the Project

### 1. Clone the Repository

```bash
git clone https://github.com/YourUsername/Airline-Reservation-System.git
```

### 2. Navigate to the Project Folder

```bash
cd Airline-Reservation-System
```

### 3. Install Required Packages

```bash
pip install -r requirements.txt
```

*(Skip this step if your project does not use external libraries.)*

### 4. Configure MySQL

* Import `SQL_Database.sql`
* Update database username, password, and database name in your project files.

### 5. Run the Project

```bash
python app.py
```

> **Note:** Replace `app.py` with your actual main file if it has a different name.
