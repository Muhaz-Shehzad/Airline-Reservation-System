# Airline Reservation System

## Semester Project

The Airline Reservation System is a desktop/web-based application developed using Python and MySQL. It allows users to search flights, book tickets, manage passenger information, and enables administrators to manage flights and reservations efficiently.

## Developed By

**Muhaz Shehzad**

## Technologies Used

- Python
- HTML
- MySQL
- Visual Studio Code

## Features

- User Login
- Admin Login
- Flight Management
- Passenger Registration
- Flight Search
- Ticket Booking
- Ticket Cancellation
- Seat Availability
- Booking History
- Database Management

## Database

MySQL

Database file:

```
airline_db.sql
```

Import this SQL file into MySQL before running the project.

## Concepts Used

- Object-Oriented Programming
- CRUD Operations
- Database Connectivity
- File Handling
- Exception Handling

## How to Run

1. Install Python.

2. Install MySQL.

3. Import

```
airline_db.sql
```

into MySQL.

4. Update database credentials in the Python project.

5. Run

```bash
python app.py
```
